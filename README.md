# user-management-api-challenge

playwright + cucumber => api test

## how to install all it needs?

```bash
npm i
```

## how to run it?

### default: local

```bash
npm test
```

### dev env

```bash
npm test:dev
```

### prod env

```bash
npm test:prod
```
